# トーン＆マナー — アーケード画像（bg_arcade）

`web/assets/bg_arcade.png` の世界観を言語化したもの。今後の画像生成・UI・発信ビジュアルを
この基準に揃えると、ゲームセンター全体の見た目が一本筋通る。

## 一言で

**80年代レトロフューチャー（シンセウェイブ／ヴェイパーウェイブ）のゲームセンター。**
夕暮れのネオン、無限に伸びるグリッド、ノスタルジーと高揚感が同居する「夜のアーケード」。

## ジャンル / 様式

- シンセウェイブ / レトロウェイブ / アウトラン調（1980s arcade futurism）
- ピクセル感は出しすぎず、**ネオン・イラスト**寄り（クリーンなベクター＋発光）

## カラーパレット

| 役割 | 色 | 近似 hex |
|------|----|---------|
| 主役（夕日・強調） | マゼンタ〜ホットピンク | `#ff3df0` / `#ff5cae` |
| 空グラデ上 | ディープパープル | `#3a1170` |
| 空グラデ下 | バイオレット〜オレンジ | `#7a2e8f` → `#ff8a5c` |
| グリッド・ライン | シアン（電光） | `#23e5ff` |
| 背景の闇 | ほぼ黒の紫紺 | `#140d17` |

- 彩度は高め、**ネオンの発光（グロー）**を必ず効かせる。
- 補色対比（マゼンタ × シアン）が骨格。中間色で濁さない。

## 構図 / モチーフ

- **中央奥に沈む（or 昇る）巨大な太陽**。横ストライプが入る定番シンセウェイブ・サン。
- **地平線**を画面中央やや上に置き、そこへ向かって**パースのかかったグリッド床**が収束。
- 左右に**アーケード筐体**を配置し、視線を中央へ導く額縁構図。中央は抜けていてUI/被写体を置ける。
- 空は上から下へ紫→ピンクのグラデ。

## ライティング / 質感

- 光源＝ネオンと太陽。全体に**発光・ブルーム**、輪郭は光る線。
- 端を軽く落とす**ビネット**で中央を締める。
- うっすら**スキャンライン／グリッド**のテクスチャでレトロCRT感。

## ムード

ノスタルジック / 高揚 / 夜 / 祝祭的 / 少しドリーミー。
「子どもの頃の薄暗いゲームセンターのワクワク」を現代のネオンで蘇らせる温度感。

## Do / Don't

- ✅ マゼンタ×シアンのネオン、発光、パースグリッド、沈む太陽、中央を抜く構図
- ✅ 高彩度・高コントラスト・クリーンな線
- ❌ パステルで淡くする／彩度を落とす／自然光・日中の明るさ
- ❌ 文字・ロゴの直接描き込み（UIは後乗せ）／写実フォト調／ごちゃつき
