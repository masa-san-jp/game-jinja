# ゲーム神社

**ゲーム神社**は、ユーザーが「作りたいゲームの気配」を選び、AIに祈ることで、実際に遊べるミニゲームを授かる生成型ゲームです。

最初にユーザーは、いくつかの選択肢から自分の欲しい遊びの要素を選びます。  
選べる要素は最大7つまでです。

たとえば、

- タイムアタック
- スコアアタック
- 対戦
- 破壊
- スピード
- ペナルティ
- 緊張と解放
- 積み上がる制約
- 連鎖
- ギリギリ回避
- 操作不能感
- 見ていて楽しい派手さ

などから、自分が惹かれるものを選びます。

その後、ゲーム神社はユーザーにこう告げます。

> あなたのゲームを出します。  
> 祈りなさい。

ユーザーは短い祈りの言葉を入力します。  
その祈りと選ばれた要素をもとに、AIがゲームコンセプト、ルール、操作、スコア、ペナルティを決めます。

最終的に出てくるのは、単なる企画書ではありません。  
**その場で遊べる小さなゲーム**です。

## 体験の流れ

1. 遊びの要素を選ぶ  
2. 最大7つまで選択する  
3. 「祈りなさい」と促される  
4. 祈りの言葉を入力する  
5. AIがゲームを神託として生成する  
6. そのまま遊べるゲームが出てくる

## 体験の核

- ゲーム企画を「選ぶ」のではなく「授かる」
- プロンプト入力を「祈り」に見立てる
- AI生成を「神託」として演出する
- 結果は説明文ではなく、実際に遊べるゲーム
- 1プレイは短く、見ていても楽しい
- 失敗、制約、スコア、速度、破壊などの遊びのイディオムを組み合わせる

## 一言で

**ゲーム神社は、遊びの気配を選び、AIに祈ると、その場で遊べるゲームが降りてくる神託型ゲーム生成体験です。**
